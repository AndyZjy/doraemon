# 快速开始

#### 安装依赖

```bash
yarn install
```

#### 本地开发启动应用

```bash
yarn dev
```

应用访问: http://127.0.0.1:7001

#### 发布模式启动应用

```bash
yarn start 

// 测试环境发布
yarn start:test
```

#### 项目构建

```bash
// 直接运行(编译文件全部在内存里面,本地开发使用)
npm start

// 编译文件到磁盘打包使用(发布正式环境)
npm run build 或者 easywebpack build prod
```
